June 2022

Needs a Linux OS

A) To launch the openFACS application:

    1)Download the tar file:
        wget https://github.com/phuselab/openFACS/releases/download/1.0.1/openFACS_Linux.tar.gz

    2)Untar the package
        tar xvf openFACS_Linux.tar.gz
    3)Make the script executable:
        chmod +x ActionUnitsFace.sh
    4)Run it:
        ./ActionUnitsFace.sh


B) To install  DeepFace :

    1) pip install deepface or conda install -c conda-forge deepface

C) Copy the "empathic-agent.py" file in this path "OpenFACS/openFACS-master/python"

D) To lauch the final program :

    1) Open terminal in the OpenFACS/LinuxNoEditor folder
    2) Run the ActionUnitsFace.sh file
    3) Run the empathic-agent.py file (press 'q' to quit)
    4) Enjoy the experience
